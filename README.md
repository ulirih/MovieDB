#### MovieDB <br>
Application use [themoviedb](https://themoviedb.org) API
<ul>
  <li>Swift, MVVM + Coordinator</li>
  <li>RxSwift, RxCocoa</li>
  <li>UIKit, Compositional Layout, DiffableDataSource</li>
  <li>Pagination loading</li>
</ul>
